Event Server

Event Fishing

Event di mana kita harus memancing untuk mendapatkan Diamond Lock. Kalian bisa cek sendiri dengan command /sellfish

Event BFG (Break For Gems)

Event BFG adalah di mana kita harus break sebuah block farmable seperti chand, lgrid, pepper, pot untuk mendapatkan Gems dan hadiah tambahan seperti Diamond Lock, Blue Gem Lock, Exp Potion

Event SSU (Spirit Storage Unit)

Event di mana kita harus memasukan sebuah ghost in jar ke dalam spirit dan kita harus mencapai % yang tinggi untuk mendapatkan Dreamcatcher Staff (DCS).dan kalian bisa gunakan /exchange untuk menukarkan dcs dengan harga:
- 1 Dreamcatcher Staff: 50 DIAMOND LOCK

Event Geiger

Event yang di mana kita harus mencari sinyal untuk mendapatkan item crystal dan kalian bisa gunakan /exchange untuk menukar item crystal dengan harga:
- Red/Green Crystal: 20 DIAMOND LOCK
- Blue Crystal: 25 DIAMOND LOCK
- Black Crystal: 1 BLUE GEM LOCK
- White Crystal: 2 BLUE GEM LOCK

Dice Game

Dice game adalah sebuah permainan menang kalah yang di mana pemenang akan mendapatkan hadiah sesuai yang diberikan oleh pembuat dice game

Password Door (PwDoor)

PWDoor adalah sebuah game dimana kita harus tebak password dari door tersebut. Jika benar memasukkan password, kita akan mendapatkan hadiah sesuai yang diberikan oleh pembuat Password Door

Event WOTD (World Of The Day)

Event di mana player harus mendesain world yang menarik dan sebagus mungkin. BacardiPS mengadakan WOTD dan mengambil 3 world terbaik dengan prize 100.000-150.000. Event berakhir tanggal 22 juni.

CARA LOGIN

ANDROID

1. Unduh "No Root Hosts GO"

2. Instal aplikasi dan buka "Hosts Editor"

3. Masukkan tautan berikut:

https://gtpshost.com/BacardiPS-G.txt

4. Ketuk "Download and Apply", lalu tekan "START"

iOS

1. Unduh aplikasi "Surge 5" dari App Store

2. Pilih "IMPORT" → "Download Profile from URL"

3. Masukkan URL berikut:

https://ios.gtpshost.com/BacardiPS-G

4. Tekan "OK" lalu "Done". Setujui kebijakan dan tekan "SETUP"

WINDOWS

1. Tekan ⊞ Win + R, ketik:

C:\Windows\System32\drivers\etc

2. Temukan file hosts, klik kanan, pilih Edit

3. Tambahkan baris berikut di akhir file:

15.235.166.218 growtopia1.com 15.235.166.218 growtopia2.com 15.235.166.218 www.growtopia1.com 15.235.166.218 www.growtopia2.com 15.235.166.218 RvLnd.here

4. Simpan file dan tutup editor

MAC

1. Buka Finder → Go → Go to Folder

2. Ketik:

/private/etc/hosts

3. Seret file hosts ke desktop untuk diedit

4. Tambahkan baris yang sama seperti Windows

5. Simpan dan pindahkan kembali ke folder asal
